#  Full Stack Budget Tracker
                          
## Description
                          
Track your income and expenses online or offline. Built using Node.JS, Javascript, HTML5, and CSS3 
                          
                          
## Installation
                          
To install necassary dependencies, run the following command:
                          
npm install
